## 🚀 Getting Started

### Prerequisites
- Node
- NPM

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kmakatsoris/Portfolio.Client.git
   cd Portfolio.Client
   ```

2. Build the application:
   ```bash
   npm install  
   ```

3. Start the application:
   ```bash
   npm start
   ```

5. Open your browser and navigate to `http://localhost:3000`.
